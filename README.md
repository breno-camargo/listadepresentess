# Lista de Presentes

App de wishlist pra mim e minha namorada. Cada um coloca o que quer ganhar e o outro ve a lista.

## Como funciona

- Login com Google (so nos dois temos acesso)
- Cada um tem sua lista editavel
- O outro ve a lista mas nao edita
- Da pra favoritar, categorizar e colocar preco
- Tem dark mode

## Stack

- Next.js 14
- Supabase (banco + auth)
- CSS Modules
- Vercel

## Rodando local

```bash
npm install
npm run dev
```

Precisa criar um `.env.local` baseado no `.env.local.example` com as chaves do Supabase.
